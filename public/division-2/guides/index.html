What the Division 2 Guides Hub Covers

The Guides hub is for practical Division 2 explanations, walkthroughs, farming routes, system breakdowns, activity preparation, and new-agent help. It supports players who need more context than a build card or tool link can provide.

Guides can help agents understand how builds work, where items come from, how systems interact, and what to prioritize when returning to the game or preparing for harder content.

Frequently Asked Questions

Are guides different from build videos?

Yes. Guides explain systems, activities, mechanics, or farming paths, while build videos focus on specific loadouts and gear setups.

Can a guide still include a build?

Yes. Some guides include build recommendations, but the main purpose is to explain a system, route, activity, or mechanic.
