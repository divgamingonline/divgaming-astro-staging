How to make changes live

  1. Select the file you want to edit, such as builds.json or resources.json.
  2. Edit the JSON, then use Validate JSON and Download edited file.
  3. In GitHub, replace the matching file inside the data/ folder and commit the change.
  4. GitHub Pages will redeploy the static site after the commit.

For real one-click saving later, this can be upgraded to Decap CMS, Supabase, Firebase, or a small Cloudflare Worker. This version intentionally stays free and GitHub Pages-compatible.
